## Installation

- clone

```bash
$ git clone git@github.com:1zumisawashun/postcoffee-lp-vue.git
$ cd postcoffee-lp-vue
```

- install

```bash
$ npm install
```

- ローカル開発用 URL を開き動作確認をする

```bash
$ npm run serve
```

http://localhost:8080/

- 上記の手順で失敗する場合 [Troubleshoot](#Troubleshoot)を確認してください
