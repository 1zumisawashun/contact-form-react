## Installation

- clone

```bash
$ git clone git@github.com:1zumisawashun/contact-form-react.git
$ cd contact-form-react
```

- install

```bash
$ npm install
```

- ローカル開発用 URL を開き動作確認をする

```bash
$ npm run dev
```

http://localhost:5173

※ 検証の導線やコメント内容などトップページに記載しておりますのでご確認ください。
