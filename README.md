## Installation

- clone

```bash
$ git clone git@github.com:1zumisawashun/eslint-template.git
$ cd eslint-template
```

- install

```bash
$ npm install
```

- ローカル開発用 URL を開き動作確認をする

```bash
$ npm run dev
```

http://localhost:5173

- 上記の手順で失敗する場合 [Troubleshoot](#Troubleshoot)を確認してください

## How to
